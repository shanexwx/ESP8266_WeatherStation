# ESP8266_WeatherStation
A weather staion with PWM output and IR emission


当前硬件版本2.0；固件版本1.4。
硬件支持的功能：oled显示（i2c）；pwm控制；亮度测量；红外发射
此版本固件可实现的功能：oled时间天气显示；舵机控制开关灯
暂未开放的功能：空调遥控，web配网；屏幕自动亮度调节
